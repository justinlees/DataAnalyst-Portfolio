# Sales DashBoard - SuperStore

This project analyses the key insights of Business like: Sales, Profits, Discounts and regional performance

## Dataset Used

Kaggle - Superstore (https://www.kaggle.com/datasets/vivek468/superstore-dataset-final?resource=download)
(Sample-Superstore)

## Tables Divided as

Orders, Customers, Products, Sales

## Key insights

- State Vs Sales : Highest sales State
- Profit Vs Sub_Category: Highest Profit and Highest loss Sub_category product
- Profit Vs Discount : Discount provision analysis on products
- Sub_category Vs Sales : Highest sales Sub_category Product

## Key Operations

- Performed SQL Joins.
- Performed Temporary Table creations.
- Usage of Aggregate functions and GROUP BY, ORDER BY clauses

## File Directory

- Dataset directory contains the original dataset.
- Tables directory have the derived tables from original dataset necessary for insights.
- Results directory contains the results of the SQL Operations(insights).
- SQL_queries_for_insights.txt file contains the SQL queries.
